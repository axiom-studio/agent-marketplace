# Database Operations

Powerful database operations for your agents.

## Features

- **Query**: Execute SELECT queries with parameterized inputs
- **Insert**: Insert records with automatic schema detection
- **Update**: Update records with flexible conditions
- **Delete**: Delete records safely
- **Transactions**: Execute multiple operations in a transaction

## Supported Databases

- PostgreSQL
- MySQL
- SQLite
- SQL Server

## Node Types

| Node Type | Description |
|-----------|-------------|
| `db-query` | Execute SQL queries |
| `db-insert` | Insert records |
| `db-update` | Update records |
| `db-delete` | Delete records |
| `db-transaction` | Execute transactional operations |

Configure database connections in your Axiom vault or environment.
