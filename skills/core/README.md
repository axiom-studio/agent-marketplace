# Core Agent Capabilities

The essential skill that provides the foundational node types for building powerful agent workflows.

## Features

- **Triggers**: webhook, cron, manual triggers
- **Control Flow**: if/else, switch, delay, loops
- **Data Operations**: transform, merge, filter, sort, aggregate
- **Actions**: HTTP requests, code execution, AI inference
- **Communication**: Slack, Discord, Teams, Email

## Installation

This skill is included by default with Axiom and is always available.

## Node Types

| Node Type | Description |
|-----------|-------------|
| `webhook` | Receive HTTP webhook requests |
| `cron` | Schedule workflows with cron expressions |
| `manual` | Manual trigger for workflows |
| `if` | Conditional branching |
| `switch` | Multi-way branching |
| `delay` | Pause workflow execution |
| `transform` | Transform data with templates |
| `http` | Make HTTP requests |
| `code` | Execute custom code (JS/Python) |
| `ai` | AI/LLM inference |
| `slack` | Send Slack messages |
| `email` | Send emails |
